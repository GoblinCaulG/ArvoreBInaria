Definição

O que é uma ABB?

Cada nó tem no máximo dois filhos. Valores menores ficam à esquerda, maiores à direita.

Clique num nó para verificar a regra.
<
Esquerda: valores menores
=
Nó: valor de referência
>
Direita: valores maiores
Conceitos

Terminologia da árvore

NóElemento que armazena um valor e ponteiros para filhos.
RaizPrimeiro nó da árvore. Não tem pai — é o ponto de entrada.
Nó internoNó que possui pelo menos um filho (não é folha).
FolhaNó sem filhos — está na ponta da árvore.
AlturaMaior distância da raiz até uma folha. Nesta árvore: 3.
ProfundidadeDistância de um nó até a raiz. Ex: nó 10 tem profundidade 2.
SubárvoreUm nó + todos os seus descendentes. Destacada em azul no diagrama.
Classificação

Tipos de árvore binária

Três classificações baseadas na estrutura dos nós e níveis.

Cheia (Full)

Todo nó tem 0 ou 2 filhos. Nenhum nó tem apenas um filho.

Completa (Complete)

Todos os níveis preenchidos, exceto possivelmente o último — que é preenchido da esquerda para a direita.

Perfeita (Perfect)

Todos os nós internos têm 2 filhos e todas as folhas estão no mesmo nível.

Operação

Como inserir um valor

Desça pela árvore comparando valores até achar um espaço vazio.
inserir()
No inserir(No no, int v) {
  if (no == null)
    return new No(v);
  if (v < no.valor)
    no.esquerda = inserir(
      no.esquerda, v);
  else if (v > no.valor)
    no.direita = inserir(
      no.direita, v);
  return no;
}
Operação

Como buscar um valor

Escolha um valor para buscar.
buscar()
boolean buscar(No no, int v) {
  if (no == null)
    return false;
  if (v == no.valor)
    return true;
  return v < no.valor
    ? buscar(no.esquerda, v)
    : buscar(no.direita, v);
}

Os três casos de remoção

        
Clique num caso para ver a animação.

      

      

        
        

          
Caso 1 · Nó folha

          
          
Sem filhos → apenas apague o nó. O pai passa a apontar para null.

        

        
        

          
Caso 2 · Um filho

          
          
O pai "pula" o nó removido e adota o neto diretamente.

        

        
        

          
Caso 3 · Dois filhos

          
          
Substitua pelo sucessor (menor da direita) e remova o sucessor original.

Conclusão

      
Árvore Binária de Busca — resumo final

      
A ABB organiza dados com uma regra simples: menor à esquerda, maior à direita. Isso permite busca, inserção e remoção eficientes quando a árvore está equilibrada.

      
O percurso em-ordem retorna os valores ordenados. A remoção com dois filhos usa o sucessor em-ordem para manter a estrutura válida.
